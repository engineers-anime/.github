## エンジニアニメとは

IT 業界に関わる方々が集まって**アニメから得た学び**について語り合う勉強会です。気軽にご参加ください！

皆さんもアニメを見てモチベーションが回復したり、頑張ろうと思えたりした経験があるのではないでしょうか。

この会では、IT 業界に関わる皆さんが「アニメに影響を受けて考えが変わった話」や「アニメから学ぶモチベーションの維持」などのトークを聴きながら交流できる場を目指しています。

懇親会では仕事の話で盛り上がるのはもちろん、アニメの話題で交流していただけると嬉しいです！
